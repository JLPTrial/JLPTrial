
# Tipos de Issues

Escolha o tipo que melhor descreve o que voce precisa. Isso ajuda a equipe a priorizar e responder mais rapido.

## Bug

Use quando algo no app não funciona como deveria.

Exemplos: botão que não responde, informação errada exibida, funcionalidade quebrada, etc.

[Ver todos os bugs reportados](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3ABug)

## Crash

Use quando o app fecha abruptamente, congela ou para de responder, exigindo reinicio forçado.

[Ver todos os crashes reportados](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3ACrash)

## Qualidade de Código

Use para reportar qualquer fluxo de código que esteja confuso ou complexo demais, discordância com boas práticas ou melhorias gerais de código que não alterem sua funcionalidade.

[Ver todas as issues de qualidade de código](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3A%22Qualidade+de+c%C3%B3digo%22)

## Performance

Use quando o app apresenta lentidão, travamentos, carregamento demorado, aumento anormal de temperatura do dispositivo ou consumo excessivo de bateria/memoria.

[Ver todos os problemas de performance](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3APerformance)

## Interface

Use para problemas visuais como elementos desalinhados, textos cortados, cores/contraste inadequados ou fluxos confusos de navegação.

[Ver todos os problemas relacionados a Interface](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3AInterface)

## Documentação

Use para reportar erros, informações desatualizadas, falta de clareza ou ausência de documentação necessaria em qualquer lugar do repositório.

[Ver todas as issues de documentação](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3ADocumenta%C3%A7%C3%A3o)

## Nova funcionalidade

Use para propor funcionalidades completamente novas que ainda não existem no app.

[Ver todas as funcionalidades solicitadas](https://github.com/JLPTrial/JLPTrial/issues?q=is%3Aissue+label%3ADemanda)
