# JLPTrial - Um aplicativo para seus estudos do exame de proficiência

Inspirado em ferramentas como o Anki, este projeto tem como principal objetivo, auxiliar estudantes que pretendem prestar o JLPT (Japanese Language Profiency Test) através de uma plataforma que centralize diversas questões de simulados separadas por seção do exame.

O aplicativo foi desenvolvido com foco:

- Atender dos níveis N5 e N4
- Permitir a identificação de dificuldades através das tags em questões
- Fornecer uma interface intuitiva e personalizavel
- Facilitar o acesso a informações estatísticas de desempenho

Inicialmente como protótipo, objetivamos que o aplicativo seja uma ferramenta de estudo individual mas com possibilidade de expansão, inserção de novos recursos e personalização para escolas ou cursos específicos.
